### 1장 후원 선교사의 관리 문제
선교위원회에서 문제가 되는 선교사 가정의 세가지 사례
* 선교사 자녀(MK)들의 교육을 위해 선교지를 이탈하여 체류 - 사역적 책무의 불이행
* 선교 사역 소식이 없음 - 사역 보고의 의무 불이행
* 선교 후원금을 파송교회에 보고 없이 모금, 현지 사역의 미진함 - 재정적 책무 및 언어적 책무 불이행
